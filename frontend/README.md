# Frontend - BTVN

## Chạy nhanh

1. Chạy backend Spring Boot ở `http://localhost:8080`.
2. Mở terminal tại thư mục `frontend`.
3. Chạy server tĩnh:

```powershell
python -m http.server 5500
```

4. Mở trình duyệt: `http://localhost:5500`.

## API mặc định

- `http://localhost:8080/api/files`

Bạn có thể đổi API Base URL trực tiếp trên giao diện.
